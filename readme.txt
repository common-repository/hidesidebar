== Description ==

Hidesidebar removes the left sidebar on the new twenty fifteen wordpress theme. This plugin was developed to help those who want to use the default wordpress twenty fifteen theme but don't want the left sidebar. It is a bit of headache for many individuals who are newbies to remove this sidebar. This plugin was developed to alleviate the burden from you.

== Installation ==

This section describes how to install the plugin and get it working.

1.    Download the plugin from http://wedevall.com/download/hidesidebar .
2.    Upload the file through the plugins tab on the wordpress dashboard
3.    After uploading you will be asked to activate the plugin. 
4.    Activate the plugin and you can refresh your front page to see changes
